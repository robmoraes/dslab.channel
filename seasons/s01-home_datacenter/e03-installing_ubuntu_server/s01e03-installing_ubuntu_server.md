# Installing Ubuntu Server

Iniciar a VM.

Durante a instalação deve ser configurado usuário, nome do servidor
e opcionalmente DHCP manual, mas vamos manter o DHCP automático
e configurar o IP fixo manualmente depois.

- Iniciar a VM no modo desassistido com rede em modo Bridge
- Configurar credenciais: u:dslab, p: dslab123
- Configurar nome do servidor: cp1
- Reiniciar após instalação e fazer login
