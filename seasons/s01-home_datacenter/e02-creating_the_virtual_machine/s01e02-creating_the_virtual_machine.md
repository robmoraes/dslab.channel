# Creating the Virtual Machine

Criar a VM com `Oracle Virtual Box` para o node principal
do cluster, o control-plane que será usado para emissão dos
tokens para vincular todos os outros nodes.

Passos:

- Download do iso do Ubuntu Server
- Criação da VM provisionando recursos iniciais
- Configurar a rede em modo Bridge
- Configurar para instalação assistida.

Serão usados recursos mínimos necessários para rodar e instalar
o Ubuntu Server.

Recursos como Memória e CPU podem ser ajustados posteriormente
mediante necessidade.
